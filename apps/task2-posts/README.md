# Task 2: Posts App (Typographic Brutalism Edition)

A React application for viewing and creating posts, designed with a high-impact "Typographic Brutalist" aesthetic.

## Features
- **View Posts**: Paginated list of posts from JSONPlaceholder.
- **Read Details**: Deep link into specific post details.
- **Create Post**: Form to submit new entries.
- **Bonus**:
  - **Caching**: Instant navigation for previously visited pages.
  - **Skeletons**: Custom loading states.
  - **Mobile Optimized**: Responsive design for all screens.
